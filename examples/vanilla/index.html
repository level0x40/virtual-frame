How it works

  • Real-time projection: Content from the source is projected into the host element with full style preservation
  • Mutation observation: Any changes in the source are automatically reflected in the projection
  • Event proxying: Interactions on the projected content trigger events in the original source
  • Canvas & video: Animated canvas and video elements are streamed in real-time
