# `@virtual-frame/solid-start`

> SolidStart integration for Virtual Frame — SSR via route queries with client-side resume

Part of [Virtual Frame](https://virtual-frame.level0x40.com) — microfrontend projection for the modern web.

## Install

```sh
npm install @virtual-frame/solid-start
# or
pnpm add @virtual-frame/solid-start
```

## Usage

See the [Virtual Frame documentation](https://virtual-frame.level0x40.com) for the API and a worked example for this integration.
